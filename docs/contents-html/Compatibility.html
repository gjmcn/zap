Compatibility

The compiled JavaScript is automatically wrapped in an IIFE and runs in strict mode.

Almost all JavaScript features used by Zap are supported by recent versions of browsers and Node.js. Features most likely to be unsupported are:

  • Nullish coalescing (??), conditional assignment (?=) and assignment from option (=:) use JavaScript's nullish coalescing operator.

  • Conditional get property (?:) and conditional call method (?|) use JavaScript's optional chaining operator.

Where appropriate, transpile JavaScript with e.g. Babel to ensure compatibility with the target environment(s).
